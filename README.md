# Scrapnews

This app scraps articles from the New York Times. It saves the articles in the database and you can add notes/comments to a specific article. You can also delete the notes or articles.

### Technologies used

- Mongoose
- Express
- Express handlebars
- Cheerio
- Axios
- HTML
